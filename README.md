# TokoGO
早稲田大学バス時刻表アプリ (iOS)
このプロジェクトは、早稲田大学（所沢キャンパス）と小手指駅間の連絡バス時刻表をリアルタイムで確認できるiOSアプリケーションです。

🚌 主な機能
スマート時刻表表示:
現在の時刻に基づき、最も関連性の高い前後5便を自動的に抽出して表示します。
次発便の背景を赤色でハイライトし、視認性を高めています。
リアルタイム・カウントダウン:
各便の出発までの残り時間を秒単位でリアルタイムに更新します。
スケジュール切り替え:
「平日」「特別」「土曜」「日曜/休日」の各ダイヤを、左右のスワイプまたはセグメントコントロールで簡単に切り替えられます。
詳細表示モード:
特定の路線をタップすることで、その日の全ダイヤを確認でき、次発便まで自動スクロールします。
サイドメニュー (Sidebar):
大学の年間カレンダー（PDF）の閲覧や、MyWasedaへのクイックアクセスが可能です。

🏗 画面構成
RootViewController
アプリの基盤。サイドメニューとメインコンテンツの切り替えを管理します。
BusTimetablePageViewController
各ダイヤ（平日/休日等）をページとして管理し、上部に切り替えUIを提供します。
BusRoutesContainerViewController
1つの画面に「往路」と「復路」の2つのリストを垂直に配置（UIStackView使用）します。
BusListViewController
特定路線の直近ダイヤを表示するテーブルビュー。タイマーによる更新を制御します。
BusCell
各バス便のカスタムセル。時間、行き先、カウントダウン、ハイライト描画を担当します。
DetailedTimetableViewController
全便リスト表示用。次発便の自動フォーカス機能を備えています。
PDFViewerViewController
大学の学年暦（PDF）を表示するための汎用ビューアです。

🛠 技術スタック
Language: Swift 5.0+UI 
Framework: UIKit (Programmatic UI / No Storyboard)
Architecture: MVC-like (Controller-driven layout)
Data Handling: JSON (BusRouteモデル経由でのパース)
Auto Layout: NSLayoutConstraint (コードによる制約定義)

📂 データ構造
バスの時刻データは busRoutes.json に格納されており、BusDataLoader によって起動時に一括読み込みされます。
HH:mm 形式の文字列を Date オブジェクトに変換して処理しています。
日付を跨ぐ運用には対応しておらず、当日内の時刻として計算されます。

📝 注意事項
このアプリは連絡バスの公式時刻表データに基づき作成されていますが、交通状況による遅延などは反映されません。
大学の学年暦を表示するには、プロジェクトのBundle内に 早稲田大学2025年カレンダー.pdf が含まれている必要があります。

💡 开发者メモ
このアプリの最大の特徴は、単なる「リスト表示」ではなく、「今、どのバスに乗れるのか」をユーザーが瞬時に判断できるように設計されている点です。UIStackView による動的なレイアウト調整と、1秒ごとに実行される Timer 処理により、常に最新の情報が保たれます。
